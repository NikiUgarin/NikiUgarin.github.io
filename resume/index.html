Мое мини приложение

Anime Tracker (NU)

Данное приложение представляет собой мини-окно, отображающееся поверх других приложений, с полупрозрачным фоном и возможностью перемещения на любую часть экрана. С помощью приложения можно отслеживать курсы популярных валют (USD, EUR, CNY), на основе данных Центробанка РФ (обновляются каждый день). Также при нажатиина кнопку "System" можно сменить отображение с курсов валют на состояние системы и отображение времени. Во время работы приложения вас будет сопровождать милейшая анимация. Закрыть приложение можно с помощью контекстного меню, которое открывается нажатием ПКМ по любому месту в приложении. Дополнительно: при первом запуске приложение добавляется в автозагрузку, есть возможность посмотреть характеристики компьютера.
